## ⌨️ Niri Keybinds

### Applications and System

| Keybind | Action |
|---|---|
| `Mod + Enter` | Open Kitty terminal |
| `Mod + D` | Open Fuzzel application launcher |
| `Mod + O` | Open Niri overview |
| `Mod + W` | Close the focused window |
| `Mod + Shift + S` | Take a screenshot |

### Window Focus

| Keybind | Action |
|---|---|
| `Mod + H` | Focus the window to the left |
| `Mod + J` | Focus the window below |
| `Mod + K` | Focus the window above |
| `Mod + L` | Focus the window to the right |

### Swap and Consume Windows

| Keybind | Action |
|---|---|
| `Mod + Shift + H` | Swap the window or column to the left |
| `Mod + Shift + L` | Swap the window or column to the right |
